```yaml

  - cron: '0 0 * * *'
```

1. **代码检出**: 自动获取最新代码

## 🔧 使用方法

### 1. GitHub部署 (推荐)
```bash
# 1. Fork或克隆仓库
git clone https://github.com/lsn7139a-art/ai-news-mcp.git

# 2. 配置环境变量
# 在GitHub仓库Settings > Secrets中添加必要的secrets

# 3. 推送到GitHub
git push origin main

# 4. 启用GitHub Actions
# 在仓库Actions页面启用工作流
```

### 2. 本地运行
```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑.env文件填入真实配置

# 3. 运行程序
npm start
```
