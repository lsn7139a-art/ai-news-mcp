## 📋 使用说明

### 快速启动
```bash
# Windows用户
quick-start.bat

# Linux/Mac用户
./quick-start.sh

# 或手动执行
cd ai-news-mcp
npm install
npm start
```

### 配置邮箱
编辑 `.env` 文件：
```env
EMAIL_HOST=smtp.example.com
EMAIL_PORT=587
EMAIL_USER=your-email@example.com
EMAIL_PASS=your-password
RECIPIENT_EMAIL=1372943709@qq.com
```
